자사 솔루션 OKMS 리뉴얼

일반형 웹사이트였던 자사솔루션 OKMS의 반응형 웹 리뉴얼 퍼블리싱 업무를 담당하였습니다. 해당 작업은 Vue.js 학습을 위해 HTML5, CSS3로 작업했던 결과물을 개인적으로 Vue.js로 재구현한 작업물입니다. 또한 스터디 목적으로 다양한 기술을 적용하여 작업 효율을 향상하기 위해 PostCSS 및 Tailwind CSS도 활용해 보았습니다.

SCSS변수를 이용하여 테마변경

고객사에게 솔루션을 제공할 때, 고객사의 브랜드 컬러를 고려한 컬러 수정 및 버튼 크기, 그림자와 같은 다양한 UI 수정 요청이 있었습니다. 이전에는 CSS를 일일이 찾아 수정해야 했기 때문에 작업 효율이 낮았습니다. 이런 문제를 해결하기 위해 SCSS 변수를 활용하여 테마 변경 및 향후 UI 수정 요청을 효율적으로 처리할 수 있게 작업하였습니다.
